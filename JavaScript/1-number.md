throw new Error('입력값이 잘못되었습니다.');

function isPositive(x,y,z){

### 문제 3

세 수 `min`, `max`, `input`을 입력받아, 다음과 같이 동작하는 함수를 작성하세요.
- `min`보다 `input`이 작으면, `min`을 반환합니다.
- `max`보다 `input`이 크면, `max`를 반환합니다.
- 아니면 `input`을 반환합니다.

예:
```
limit(3, 7, 5); -> 5
limit(3, 7, 11); -> 7
limit(3, 7, 0); -> 3
```

* 방법 1

```js

function limit(min,max,input){
  if (min > input) {   
    return min;   // `min`보다 `input`이 작으면, `min`을 반환
  } else if (max < input) {
      return max;  // `max`보다 `input`이 크면, `max`를 반환합니다.
    } else {
      return input; // 아니면 `input`을 반환합니다.
    }
}

// console.log(limit(3,7,5)); 반환값 : 5
// console.log(limit(3, 7, 11)); 반환값 : 7
// console.log(limit(3, 7, 0)); 반환값 : 3


```

### 문제 6

자연수를 입력받아, 그 수의 모든 약수를 출력하는 함수를 작성하세요.

* 접근방법

  + 자연수 : for문 사용해서 초기값은 1(입력되는 수 n을 나누는 자연수), 범위는 i(나누는 값)가 입력되는 값(n)보다 작거나 같다, 1씩 증가
  + 모든 약수 : n(입력되는 값이) i로 나누어졌을 때 나머지가 0일때
  + 출력 : console.log()
  + 함수 호출
  
```js

function divide(n){
  for(i=1; i<=n; i++){
    if(n % i === 0){
      console.log(i);
    }
  }
}

divide(20);  // 출력값 : 1,2,4,5,10,20 (20의 약수)

```
